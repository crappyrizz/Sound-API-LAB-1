# Sound-API-LAB-1

A simple web audio player built with HTML, CSS, and JavaScript. This project demonstrates custom audio controls and a waveform-style visualizer while letting users select and remove their own audio files.

## Features

- Custom playback controls: play, pause, mute, loop
- Volume and playback speed adjustment
- Seek slider with current time and duration display
- Audio file selection from the local device
- Remove audio file option with dynamic UI behavior
- Simple waveform visualizer using the Web Audio API

## Keyboard Shortcuts

Space Play / Pause
←  Skip backward 5 seconds 
→  Skip forward 5 seconds |
M  Mute / Unmute toggle |

## How to use

1. Open `index.html` in a browser.
2. Click `select audio file` to choose a local audio track.
3. Use the player controls to play, pause, mute, or loop the audio.
4. Adjust volume, speed, and seek through the track.
5. Click `remove` to clear the loaded audio and reset the interface.

## Files

- `index.html` — app structure and audio controls layout
- `style.css` — styles for the player UI and visualizer
- `style.js` — audio logic, file loading, visualizer, and UI updates
